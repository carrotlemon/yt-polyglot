## Long description

**YT Polyglot adds multiple language subtitles to YouTube — at the same time.** Pick two, three, or more languages and they all display simultaneously, stacked above the player so you can read every line at once.

Built for language learners, translators, bilingual viewers, and anyone who wants dual or multi-language subtitles on YouTube without switching tracks back and forth.

### What it does

Show YouTube subtitles in multiple languages on screen at once. Combine:

- The video's original captions
- YouTube's auto-generated (speech-to-text) captions
- Any of YouTube's auto-translations — over 130 target languages

Each language gets its own row, neatly stacked above the YouTube player controls. The first language you pick sits at the bottom; the next stacks above it, and so on. No overlap, no blocked video.

### Perfect for

- **Language learning** — watch in your target language with your native language as backup. Great for learning Korean (한국어), Japanese (日本語), Chinese (中文), Spanish, French, German, Italian, Portuguese, Russian, Arabic, Hindi, Vietnamese, Thai, Tagalog, Turkish, and 130+ other languages.
- **K-dramas, anime, and foreign film** — pair the original audio with English (or any language) subtitles. Korean + English, Japanese + English, Chinese + English, Spanish + English, and every other combination.
- **Bilingual households** — show YouTube content in two languages at once so both speakers can follow.
- **Translation comparison** — see how a video translates into multiple languages side by side. Useful for translators, linguists, and language teachers.
- **ESL, EFL, ELL** — English learners can watch English videos with native-language captions for comprehension support.
- **Polyglots and casual learners** — keep multiple languages on screen for passive exposure across your YouTube watching.

### Features

- **Multiple subtitles displayed simultaneously** — no more switching subtitle tracks or watching the same video twice.
- **130+ languages** supported via YouTube's translation system.
- **Smart vertical stacking** — subtitles never overlap each other or block the video frame.
- **Search bar** — instantly find any language even when there are 150+ to choose from.
- **"Original only" filter** — hide auto-translated tracks and show only captions actually uploaded by the creator (or YouTube auto-generated).
- **Optional language labels** — toggle a small tag next to each subtitle row so you always know which subtitle is which.
- **Remembers your picks** between videos and sessions.
- **Free and open-source.**
- **Lightweight** — no background processes, no analytics, no remote servers.

### How it works

YT Polyglot uses YouTube's own subtitle and translation system. The extension does not run any translation itself — translations come from YouTube/Google's machine translation, exactly the same translations YouTube's built-in "auto-translate" option would produce. The difference is that YouTube only lets you display one subtitle track at a time; YT Polyglot fetches multiple and displays them together on the player.

### Privacy

- No analytics
- No third-party servers
- No tracking
- Nothing leaves your browser

The only network requests are direct calls to YouTube's caption API — the same calls YouTube's own player makes.
