YT Polyglot — Privacy Policy

Last updated: 2026-06-11

Summary

YT Polyglot does not collect, store, transmit, sell, or share any personal information. The extension operates entirely within your browser.

What the extension stores

The extension uses Chrome's chrome.storage.local API to save your preferences locally on your device:

  • The list of subtitle languages you have selected as active
  • Whether the "Original only" filter is on
  • Whether the "Show language labels on subtitles" setting is on

This data never leaves your browser. It is not synced to any cloud service by the extension. (Your browser's own sync settings, if enabled, are controlled by you in your browser settings, not by this extension.)

What the extension does on the network

YT Polyglot makes network requests only to youtube.com, using YouTube's own caption and auto-translation APIs — the same APIs YouTube's built-in player uses. No requests are made to any other domain.

The extension performs no translation itself. Auto-translated subtitle tracks are translated by YouTube/Google on YouTube's servers; this extension only displays the result.

What the extension does not do

  • No analytics — no Google Analytics, no Mixpanel, no Sentry, nothing.
  • No third-party servers — there is no backend.
  • No tracking — the extension does not observe what you watch, search, or click.
  • No advertising.
  • No selling or sharing of user data — there is no user data to sell.
  • No remote code execution — all code is bundled in the extension package.

Permissions

  • storage — to remember your selected subtitle languages between sessions.
  • host_permissions for *.youtube.com — to read the page's caption track list and overlay subtitle elements onto the YouTube player. The extension does not run on any other site.

Children

YT Polyglot does not knowingly collect data from children or any other users, because it does not collect data from anyone.

Changes to this policy

If this policy changes, the "Last updated" date above will be revised. The extension's data-handling behavior is unlikely to change because it currently handles no user data.

Contact

Questions or concerns: please open an issue on the project's source repository, or contact the maintainer at the email shown on the Chrome Web Store listing.
